# Lit + Vite

This template provides a minimal Lit setup powered by Vite for building lightweight web components with JavaScript.

## Usage

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

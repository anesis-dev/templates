# Preact + TypeScript + Vite

This template provides a minimal Preact setup powered by Vite for building fast web applications with TypeScript.

## Usage

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
